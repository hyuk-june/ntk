◎ 설치방법 (Install)

1. 압축을 해제합니다.

2. 압축해제된 디렉터리들을 그누보드가 설치된 경로에 업로드합니다.

3. NTK빌더 테마를 선택합니다.

   그누보드 관리자 -> 기본설정 -> 테마설정 -> NTK빌더 선택
   

   
◎ 참고사항

1. 기본데이타를 입력하지 않길 원하시면 /theme/bart/theme.config.php 파일에서 아래부분을 수정해주세요


define('_BT_DATA_', true);

위 부분을 아래와 같이 false 로 변경해 주세요.

define('_BT_DATA_', false);
